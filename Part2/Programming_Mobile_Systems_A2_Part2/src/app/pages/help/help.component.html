Help Center

1. Frequently Asked Questions (FAQ)

Q: How to add a new item?

A: Click the "Add New Item" button at the top of the page, fill out the form, and save.

Q: How to edit / delete an item?

A: Click "Edit" to modify or "Delete" to remove an item on the right side of each row. Deletion requires confirmation.

Q: How to batch delete items?

A: Check the checkboxes on the left and click "Batch Delete" to remove multiple items.

Q: How to search for items?

A: Enter the item name or select filter conditions in the search area, then click Search.

Q: Why can't I add an item?

A: ID must be unique; Name and ID are required; quantity and price cannot be negative.

Q: Will data be saved permanently?

A: No. Data will be reset when you refresh or close the page.

2. Troubleshooting

Q: Page unresponsive / buttons not working

A: Refresh the page; check if any popup window is blocking operations.

Q: Invalid number input / error

A: Quantity and price must be valid numbers; letters, symbols, and negative values are not allowed.

Q: No search results found

A: Check your keywords; clear the search box to show all data.

Q: Cannot edit an item

A: Close other popups first, then click Edit; make sure the item has not been deleted.

3. Usage Tips

1. All operations only apply to the current session. Data resets when the page is closed.

2. Red validation messages indicate invalid inputs. Please correct them as prompted.
